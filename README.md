# E-Ticketing Helpdesk — Frontend (Flutter)

Aplikasi mobile untuk pelaporan tiket helpdesk.

## Cara Menjalankan
1. Clone repo
2. `flutter pub get`
3. Ganti `baseUrl` di `lib/utils/constants.dart` dengan IP backend
4. `flutter run`

## APK
APK tersedia di folder `apk/`.
